QAssemble
=========
QAssemble，轻量级汇编IDE V1.0

使用说明
--------
* 下载或克隆本项目
* 打开QAssemble中的QAssemble.exe开始汇编之旅

注意事项
--------
* 如果您的电脑安装过DOSBox，需要将DOSBox自带的conf文件中的[autoexec]项清空
* 编译和运行按钮在软件的右上角
* 本版本暂不支持调试
* 软件下方的Building Log栏不支持关闭
